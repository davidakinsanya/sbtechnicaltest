## Logging In

A list of extensions you can you to create a dummy email are: 
"@gmail.com", "@gmail.co.uk", "@yahoo.com", "@yahoo.co.uk",
"@hotmail.com", "@hotmail.co.uk", ".ac.uk"

A dummy password must:

Contain at least 8 characters
A unique character
An uppercase letter
